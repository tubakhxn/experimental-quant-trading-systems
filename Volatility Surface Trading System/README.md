# Volatility Surface Trading System

## dev/creator = tubakhxn

---

## Project Overview
This project is a **Volatility Surface Trading System** that demonstrates:
- Synthetic generation of financial time series data (time, volatility, price)
- Polynomial regression modeling of price as a function of time and volatility
- 3D surface visualization of the volatility surface using matplotlib
- Gradient-based trading signal generation and PnL simulation
- Clean, automatic plotting of all results

**This project is for educational and research purposes only.**

## How to Fork
1. Click the "Fork" button at the top right of this repository on GitHub.
2. Clone your forked repository to your local machine:
   ```bash
   git clone https://github.com/YOUR_USERNAME/Volatility-Surface-Trading-System.git
   ```
3. Install dependencies:
   ```bash
   pip install numpy matplotlib scikit-learn
   ```
4. Run the script:
   ```bash
   python vol_surface_trading_system.py
   ```
