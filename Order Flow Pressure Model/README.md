# Order Flow Pressure Trading Model

## Project Overview
This project simulates an order flow pressure trading model using synthetic data. It generates buy/sell volumes, calculates order flow imbalance, models price movement with regression, and implements a simple trading strategy. The script visualizes price, imbalance, PnL, and a 3D surface of time, imbalance, and price. 

**Purpose:**
- For educational and research purposes only. Not for live trading or financial advice.

## How to Fork
